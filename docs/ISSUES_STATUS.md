# 課題対応表

開発中に発見された課題とその対応状況を記録する。

## 凡例

| status | 意味 |
|--------|------|
| 🔴 未対応 | まだ着手していない |
| 🟡 対応中 | 作業中 |
| ⏸️ 保留 | 意図的に後回し |
| 🟢 完了 | 解決済み |

| 優先度 | 意味 |
|--------|------|
| 高 | 開発がブロックされる、すぐに対応必要 |
| 中 | 実装フェーズで対応すればOK |
| 低 | 体裁・改善、いつでも対応可 |
