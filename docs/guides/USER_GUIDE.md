# 利用者ガイド

このガイドは、Discord Business Assistantを利用する請負者向けの説明資料です。

## 1. Discordとは

Discordは、チャット・通話ができるコミュニケーションツールです。
スマートフォン・PC両方で利用できます。

## 2. インストール方法

### スマートフォン

1. App Store（iPhone）またはGoogle Play（Android）を開く
2. 「Discord」を検索
3. インストール
4. アカウント作成（メールアドレスが必要）

### PC

1. https://discord.com/download にアクセス
2. お使いのOSに合わせてダウンロード
3. インストール
4. ログイン

## 3. サーバーへの参加

依頼者から送られた招待リンクをクリックすると、サーバーに参加できます。

## 4. 基本的な使い方

### メッセージ送信

1. 左側のチャンネル一覧からチャンネルを選択
2. 下部の入力欄にメッセージを入力
3. Enterキー（またはスマホの送信ボタン）で送信

### ファイル送信

1. 入力欄の左にある「+」ボタンをクリック
2. 「ファイルをアップロード」を選択
3. 送信したいファイルを選択

### 通話

1. 音声チャンネル（スピーカーマーク）をクリック
2. 通話に参加
3. 退出する時は「切断」をクリック

## 5. チャンネルについて

サーバー内には複数のチャンネル（部屋）があります。

| チャンネル | 用途 |
|------------|------|
| #general | 一般的なやり取り |
| #技術相談 | 技術的な質問・相談 |
| #事務連絡 | 事務的な連絡 |

※チャンネル構成は依頼者によって異なります。

## 6. AI Botについて

このサーバーには、AIアシスタント（Bot）が参加しています。

### Botの役割

- 会話の記録・保存
- 依頼者への通知
- Botがあなたに直接話しかけることは基本的にありません

### 録音について

通話は記録される場合があります。
録音されている場合は事前にお知らせします。

## 7. コマンド一覧

| コマンド | 内容 |
|----------|------|
| /summary | 直近の会話を要約 |
| /search | 過去メッセージ検索 |
| /remind | リマインダー登録 |
| /record | 通話録音の開始/停止 |
| /transcribe | 録音の文字起こし |
| /save | 最新の添付ファイルをGoogle Driveへ保存 |

`/save` は必要に応じてフォルダ指定ができます（例: `/save client-a/design`）。

## 8. 困った時は

- アプリが動かない → 再起動してみてください
- 招待リンクが使えない → 依頼者に連絡してください
- その他 → 依頼者にお問い合わせください

## 9. 注意事項

- このサーバーでのやり取りは記録されます
- 他の請負者の情報は見ることができません
- 不明点があれば遠慮なく依頼者にご質問ください
