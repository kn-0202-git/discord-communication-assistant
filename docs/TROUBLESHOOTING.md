## 環境構築

### uv: command not found

#### 症状
`uv` コマンドが見つからない

#### 原因
uvがインストールされていない、またはPATHが通っていない

#### 解決策
```bash
# macOS/Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# PATHを再読み込み
source ~/.zshrc
```
